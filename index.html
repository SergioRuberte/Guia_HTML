Etiqueta <html>

Definición: Delimita el contenido del documento
Etiquetas: Apertura y cierre opcionales
Atributos: dir, lang

dir: ltr(izquierda a derecha), rtl(derecha a izquierda)
Indica el sentido de lectura de la página.

lang: Indica el idioma del documento.

Contenido:

<head> y <body>

Etiqueta <head>

Definición: Delimita la cabecera del documento
Etiquetas: Apertura y cierre opcionales
Atributos: dir, lang, profile

profile: URI (dirección web completa del recurso)
Indica dónde están los perfiles para interpretar los elementos meta.

Contenido:

<title>,<base>,<meta>,<link>,<object>,<script> y <style>
Solamente elemento <title> es obligatorio

Etiqueta <body>

Definición: Delimita el cuerpo del documento
Etiquetas: Apertura y cierre opcionales
Atributos: dir, lang, id, class, title, style, onload, onunload

id: permite asignar un nombre único a un elemento

class: permite asignar un nombre de clase a un elemento

title: agrega un comentario a un elemento (se muestra al situar el ratón sobre el elemento)

style: aplica información de estilo a un elemento

onload: especifica el evento que ocurrirá al finalizar la carga

onunload: especifica el evento que ocurre al eliminar un elemento de una ventana o marco

Elementos de la cabecera

Etiqueta <title>| Etiqueta <base> | Etiqueta <meta> | Etiqueta <link>| Etiqueta <script> | Etiqueta <style> |

Etiqueta <title>

Definición: Indica el título en la pestaña del navegador.
Etiquetas: Apertura y cierre obligatorias
Atributos: dir, lang
Ejemplo:

<head>
<title> Titulo de la página </title>
</head>

Etiqueta <base>

Definición: Indica la dirección raíz el sitio web, la cual permiete resolver las direcciones relativas.
Etiquetas: Sin etiqueta de cierre.
Atributos: href

href: se usa siempre para indicar la dirección del documento

Ejemplo:

<base href="http://www.paginaweb.com">
En caso de tener la direccion relativa "/Articulo1.html" se completaría como "http://www.paginaweb.com/Articulo1.html"

Etiqueta <meta>

Definición: Indica un conjunto de propiedades generales del documento.
Etiquetas: Sin etiqueta de cierre.
Atributos: lang, dir, name, content, http-equiv

name: Indica el nombre de una propiedad

content: Contiene el valor de la propiedad indicada en name

http-equiv: Indica al navegador una propiedad en forma de http como si la hubiera generado el servidor web

Ejemplo:

<meta name="descripcion" content="Guia HTML">
<meta name="date" content="23/11/2016">

Etiqueta <link>

Definición: Vincula otro documento indicado por href.
Etiquetas: Sin etiqueta de cierre.
Atributos: lang, dir, class, id, style, title, charset, href, hreflang, type, rel, rev, media

charset: Establece el conjunto de caracteres para el documento

href: Indica la URI del documento vinculado

hreflang: Indica el lenguaje del documento vinculado

type: Indica el tipo de documento vinculado

rel: Estable la relacion origen-destino entre los documentos

rev: Establece un vínculo inverso. (Destino-Origen)

media: Indica el medio al que se van a proyectar los datos.

Etiqueta <script>

Definición: Inserta un script en el documento.
Etiquetas: Las etiquetas de apertura y cierre son obligatorias.
Atributos: charset, type, src

charset: codificacion de caracteres incidada por src

type: Indica el lenguaje de programación utilizado

src: indcia la URI donde está ubicado el script externo

Ejemplo:

Interno: <script type="text/javascript">> . . . . </script>
Externo: <script type="text/javascript" src="https://www.google.com/jsapi"> </script>

Etiqueta <style>

Definición: Permite insertar una hoja de estilo.
Etiquetas: Las etiquetas de apertura y cierre son obligatorias.
Atributos: lang, dir, type, media, title

type: Indica el tipo de lenguajes

type: Indica el tipo de medio al que se dirige la información

title: Texto descriptivo del estilo

Ejemplo:

<style type="text/css">
body {
margin-left: 40px; margin-right: 40px;
margin-top: 40px;
}
</style>

Etiqueta <p>

Definición: Establece una agrupación de líneas de texto en un párrafo.
Etiquetas: Etiqueta de cierre opcional.
Atributos:
Ejemplo:

Etiqueta <br>

Definición: Permite introducir saltos de línea explícito dentro de un mismo párrafo.

Etiqueta <hr>

Definición: Permite separar texto con una línea horizontal visible dentro de un mismo párrafo.
Etiquetas: Sin etiqueta de cierre.

Definición: Permiten establecer los diferentes estilos de fuente.
b: Negrita i: itálica big: Tamaño grande small: Tamaño pequeño
s y strike: Tachado tt: Teletipo u: Subrayado
Etiquetas: Sin etiqueta de cierre.

Etiqueta <font>

Definición: Permite establecer el tamaño, el color y la fuente para el texto.
Etiquetas: Apertura y cierre obligatorias.
Atributos: size, color, face

Etiqueta <pre>

Definición: Permite introducir texto con un formato propio.
Etiquetas: Apertura y cierre obligatorias.
Atributos: width

width: Indica el ancho del bloque

Manejo de enlaces

Etiqueta <a>

Definición: Se utiliza para establecer el origen o el destino de un hipervínculo.
Etiquetas: Apertura y cierre obligatorias
Atributos: charset, type, name, href, hreflang, rel, rev, accesskey, shape, coords, tabindex, onfocus, onblur

name: Permite nombrar un marcador, que será el destino del enlace

href: Indica la URI destino del hipervínculo

accesskey: Asigna una tecla única de acceso al hipervínculo

shape: Establece la forma de la imagen que contiene el hipervínculo

coords: Estable la posición

target: Estable el marco de destino. (Valores:_blank, _self, _parent, _top)

tabindex: Establece el número de orden

Imagenes

Etiqueta <img>

Definición: Sirve para insertar una imagen determinada por el atributo src.
Etiquetas: Sin etiqueta de cierre.
Atributos: src, alt, name, height, width, usemap, ismap

src: Indica la URI de la imagen.

alt: Texto descriptivo de la imagen. Aparece en caso de no poder visualizar la imagen.

name: Identifica la imagen para su manejo con hojas de estilo o scripts

height, width: altura y anchura con la que el navegador visualizará la imagen

usemap: Asocia a la imagen un mapa de imagenes.

ismap: Permite definir un mapa de imágenes.

Ejemplo:

Etiqueta <map>

Definición: Sirve para crear un mapa de imágenes en el lado cliente.
Etiquetas: Apertura y cierre obligatorias.
Atributos: name

Etiqueta <area>

Definición: Permite especificar una región geométrica del mapa y el vínculo asociado a esa región.
Etiquetas: Sin etiqueta de cierre.
Atributos: shape, coords, href, alt, tabindex, accesskey, onfocus, onblur

shape: Especifica el tipo de región con los valores: default, rect, circle, poly

coords: Posición en palntalla, valores de longitud partiendo de la esquina superior izquierda de la pantalla y separas por comas.

Ejemplo:

Tablas

Etiqueta <table>

Definición: Inserta una tabla de tamaño variable.
Etiquetas: Apertura y cierre obligatorias.
Atributos: summary, width, border, frame, rules, cellspacing, cellpadding

summary: Resumen del contenido y estructura (para buscadores)

width: Anchura total de la tabla en pixeles (px) o porcentaje (%)

frame: Especifica qué lados de la tabla serán visibles

void: Ninguno above: Arriba Below: Abajo lhs: Izquierda rhs: Derecha box: Todos

rules: Especifica qué líneas de division serán visibles

none: Ninguna groups: Sólo entre grupos de filas o columnas rows: Sólo entre filas cols: Sólo entre columnas all: Todas

cellspacing: Especifica el espacio entre celdas

cellpadding: Especifica el espacio entre el borde de una celda y su contenido.

Ejemplo:

Etiqueta <tr>

Definición: Crea una fila dentro de una tabla.
Etiquetas: Etiqueta cierre opcional.
Atributos: align, valign, char, charoff

Etiquetas <td> <th>

Definición: Sirven para crear una celda bien dentro de una fila, o bien dentro de la cabecera.
Etiquetas: Etiqueta cierre opcional.
Atributos: abbr, axis, headers, scope, rowspan, colspan, align, valign, char, charoff

rowspan: Indica el número de filas que ocupa cada celda

colspan: Indica el número de columnas que ocupa cada celda

Etiqueta <caption>

Definición: Permite poner título en una tabla.
Etiquetas: Apertura y cierre obligatorias.

Etiquetas <thead> <tbody> <tfoot>

Definición: Permite agrupar filas en la cabecera, el cuerpo, o el pie de la tabla.
Etiquetas: Apertura y cierre obligatorias.

Etiqueta <colgroup>

Definición: Permite crear grupo de columnas.
Etiquetas: Apertura y cierre obligatorias.
